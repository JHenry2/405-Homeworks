The HTML Elements I know

  1. Paragraph (p)
    1. Mozilla Doc Link
    2. Display: Block
    3. Attributes
      1. The paragraph element uses the global attributes
  2. Header (H1, H2, H3, etc)
    1. Mozilla Doc Link
    2. Display: Block
    3. Attributes
      1. The H1-H6 elements use the global attributes
  3. Ordered List (ol)
    1. Mozilla Doc Link
    2. Display: Block
    3. Attributes
      1. Reversed: Specifies that the list is in reverse order
      2. Start: Specifies the numeber the list will start counting from
      3. Type: Specifies the numbering (or lettering) type the list is sorted by
  4. Anchors (a)
    1. Mozilla Doc Link
    2. Display: Inline
    3. Attributes
      1. Href: Used to specify where the link points to
      2. Download: Tells the browser to download the url pointed to
      3. Target: specifies where to display a linked url
  5. Strong (strong)
    1. Mozilla Doc Link
    2. Display: Inline
    3. Attributes
      1. The Strong element uses the global attributes
  6. Image (img)
    1. Mozilla Doc link
    2. Display: Inline
    3. Attributes
      1. Src: The url of the image
      2. Alt: A text description of image for accessability
      3. Height: The height of the image in pixels

Vocab Definitions

  • HTTP
    1. Rules used to transfer data over the internet.
    2. Source
  • HTML
    1. Language read by the browser to create the content of a web page
    2. Source
  • Element
    1. Contains an opening and closing tag (alligators with words between), and can include content in between the two tags.
    2. Source
  • Tag
    1. The alligators with words inside
    2. Source
  • Block (CSS)
    1. Something that is displayed as a seperate "block" or line of content.
    2. Source
  • Doctype
    1. Starting line that says a document is an html document
    2. Source
  • Head
    1. A special element that has tags like meta and title nested inside of it. Seperate from the body element, which has the real content nested inside.
    2. Source
  • Attribute
    1. "extentions" inside an opening tag
    2. Source
  • Boolean
    1. A true/false on/off status
    2. Source
  • Dom
    1. Document Object Module. A tree created by the browser that shows which elements are children of which other elements
    2. Source
  • CSS
    1. Stands for Cascading Style Sheets. It is a language used to change how a website looks visually.
    2. Source
  • Property (CSS)
    1. A part of CSS that can have a value assigned to it. background color would be an example of a property
    2. Source
  • Selector (CSS)
    1. Part of the CSS language used to define which HTML elements will be affected by a declaration.
    2. Source
